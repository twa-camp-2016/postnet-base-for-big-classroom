#POST-NET

###命令行界面版Postnet.
JS做前端开发; node.js做后台开发,未使用面向对象技术
##运行步骤：
    1.git clone git@github.com:Demo-tong/postnet-Nodejs.git
    2.npm install安装依赖
    3.node transformer/server.js开启服务器
    4.新打开一个终端，输入node transformer/app.js运行程序
    5.根据提示执行该程序
    6.npm test 运行jasmine进行测试
##Notice:在测试时应打开服务器
